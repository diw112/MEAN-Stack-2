# MEAN-Stack-2

# Get Started
# Install Nodemon
https://nodemon.io

  1. Delete Folder Client
  3. Angular CLI => `ng new client`
  4. `cd client/`
  5. edit file package.json
  
  ` "start": "ng serve"` Change to `"start": "ng build --dev" ` 

  6. `nodemon --watch src/app/*.*.* --watch src/app/*/*.*.*`

# Start Mongodb
`mongod`

# Start Server
`nodemon server.js`
