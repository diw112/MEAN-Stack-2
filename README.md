# MEAN-Stack-2

# Get Started
  1. Delete Folder Client
  3. Angular CLI => `ng new client`
  4. `cd client/`
  5. edit file package.json
  ` "start": "ng serve"` Change to `"start": "ng build --dev" ` 
  6. Angular CLI => `ng build`

# Install Nodemon
https://nodemon.io

# Start Mongodb
`mongod`

# Start Server
`nodemon server.js`
